# 05 Design 设计规范（Visual + Interaction）

## 风格关键词
纸感、安静、温和、非评判。

## 色彩系统
- 背景：#F6F3EA
- 主文字：#1F2937
- 强调：#7BC4F2
- 正向：#F6C94A
- 负向：#9AB0A6
- 成功/警告/错误：#22C55E / #F59E0B / #EF4444

## 字体与层级
- 字体：Noto Sans SC + Inter
- H1: 32/40
- H2: 24/32
- Body: 16/24
- Caption: 13/20

## 关键组件
- 双滑条输入（时间、分值）
- 情绪曲线图（monotone）
- 情绪点卡片
- 复盘卡片
- 触发器标签列表

## 状态规范
- Loading：骨架屏
- Empty：引导文案
- Error：可重试按钮
- Success：保存提示 + 振动反馈（可选）

## 交互原则
- 默认值智能预填（当前小时、0分）。
- 所有关键操作 1 次点击可完成。
- 后台配置改动即时生效。
