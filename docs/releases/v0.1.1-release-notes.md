# Release Notes · v0.1.1

发布日期 / Release Date: 2026-02-11

## 中文

### 新增

- 新增完整 Docker 首次部署方案：
  - `Dockerfile`
  - `docker-compose.yml`
  - `.env.docker.example`
  - `docs/deployment-docker.md`
- 支持首次部署时自动创建并启动 PostgreSQL 容器。
- Web 服务启动前自动重试执行数据库迁移（`pnpm db:migrate`）。

### 改进

- 移除 `next/font/google` 远程字体依赖，改为系统字体策略，避免受限网络导致构建失败。
- README 增加 Docker 快速部署说明。

### 兼容性

- 保持现有 API 和功能行为不变。
- 保持 PostgreSQL + Drizzle 方案不变。

---

## English

### Added

- Full Docker first-deploy stack:
  - `Dockerfile`
  - `docker-compose.yml`
  - `.env.docker.example`
  - `docs/deployment-docker.md`
- Automatic PostgreSQL container bootstrap on first deploy.
- Automatic migration retries (`pnpm db:migrate`) before web startup.

### Improved

- Removed remote `next/font/google` dependency and switched to system-font strategy for deterministic builds in restricted networks.
- Updated README with Docker quick-start deployment steps.

### Compatibility

- No breaking API changes.
- PostgreSQL + Drizzle architecture unchanged.
