# 06 开发计划（6周）

## Week 1：工程与认证
- Monorepo + Next.js + packages 初始化
- 登录会话
- 主题与基础组件

## Week 2：打点与曲线
- 打点API + 页面交互
- 曲线可视化
- 编辑/删除
- 离线队列

## Week 3：复盘与规则AI
- 规则引擎
- 波峰/波谷识别
- 复盘文案生成

## Week 4：趋势与触发器
- 周/月聚合
- 触发器分析
- 洞察页交付

## Week 5：后台与隐私
- 指标看板
- 词典与风险词配置
- 导出/删除数据

## Week 6：质量与上线
- 单元/集成/E2E关键路径
- 性能优化
- 发布清单与监控

## 依赖关系
- 复盘依赖：打点数据 + 规则引擎
- 洞察依赖：稳定聚合查询
- 后台依赖：权限与配置存储

## 风险与应对
- 识别准确率不足 -> 词典可配置+快速迭代
- 性能波动 -> 图表轻量化+接口缓存
- 用户隐私担忧 -> 明确导出/删除能力
