# 03 术语与指标（Glossary & Metrics）

## 术语
- 情绪点（Mood Entry）：用户在某时刻的一次情绪记录。
- 波峰（Peak）：当天分值最高的情绪点。
- 波谷（Valley）：当天分值最低的情绪点。
- 触发器（Trigger）：与情绪变化显著相关的事件类别。
- 情绪标签（Tags）：由词典和分值规则推断出的标签。
- 能量消耗分（Energy Load）：综合情绪绝对值与负向时段的强度指标。

## 指标定义
- DAU（日活）：当日有任意有效行为（登录/打点/复盘）用户数。
- 当日复盘完成率：生成并查看今日复盘用户 / 当日有打点用户。
- 7日留存：D0新增用户在D7当天活跃比例。
- 周复盘触发率：近7日内访问趋势页用户 / 近7日活跃用户。
- 触发器识别覆盖率：有triggerKeys的情绪点 / 总情绪点。

## 数据口径
- 时间统一使用 ISO-8601（UTC），前端按本地时区展示。
- 同用户数据隔离，跨用户统计仅后台指标使用聚合值。
