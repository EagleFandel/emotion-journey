# 01 需求理解与目标（Requirement Brief）

## 一句话定义
情绪记录不是填表，而是“点一下 + 拉一下 + 一句话”。

## 用户价值
- 低门槛记录：15秒内完成一次打点。
- 当天复盘：自动识别波峰/波谷并给出温和反馈。
- 长期洞察：周/月趋势和高频触发器帮助用户做长期自我管理。
- 可被解释：规则型AI输出可追溯，不是黑箱结论。

## 核心场景
1. 即时打点：情绪发生后快速记录。
2. 晚间回看：结束当天时查看情绪曲线和总结。
3. 周/月复盘：发现重复触发器与高风险时段。
4. 管理调优：后台更新词典和风险词，提升识别质量。

## 非目标
- 医疗诊断与治疗建议。
- 危机干预替代方案。
- 复杂社交社区能力。

## 成功指标（MVP）
- 当日复盘完成率 >= 35%
- 7日留存 >= 20%
- 周复盘触发率 >= 40%
- 触发器识别覆盖率 >= 65%
