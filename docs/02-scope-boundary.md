# 02 范围边界（Scope Boundary）

## In Scope（首版必须）
- 账号登录（邮箱）
- 情绪打点 CRUD（时间、分值、备注）
- 当日情绪曲线（平滑）
- 今日复盘（波峰/波谷、标签、总结）
- 周/月趋势图
- 触发器识别与高频提示
- 轻量后台（词典、风险词、指标）
- 用户隐私能力（导出、删除）

## Out of Scope（延后）
- 实时多人互动
- 复杂推荐系统（个性化策略引擎）
- 原生App独立开发
- 社交分享生态

## 风险边界
- 遇到高风险语义时，仅展示安全提示，不做医疗判断。
- AI输出必须保持“建议语气”，避免确定性心理结论。
