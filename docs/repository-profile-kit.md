## 推荐描述（中文）

Emotion Journey：一个“轻记录 + 强可视化 + 规则型 AI 复盘”的情绪陪伴 Web PWA，支持趋势分析、触发器洞察、隐私导出与轻量后台。

## Recommended Description (English)

Emotion Journey is a web-first PWA for emotion tracking with lightweight input, strong visualization, and rule-based AI reflection, including trend insights, trigger analysis, privacy export, and admin tools.
