# 04 PRD（MVP）

## 产品定义
用最低操作成本记录情绪波动，并将当下体验转化为可理解的长期情绪地图。

## 功能优先级
### P0
- 登录（邮箱）
- 情绪打点
- 当日曲线
- 今日复盘
- 周/月趋势
- 触发器识别

### P1
- 离线打点队列
- 数据导出/删除
- 轻量后台

### P2
- 个性化提醒策略
- 分享卡片
- 原生App封装

## 用户流
### Flow A：快速打点
今天页 -> 点击“+打点” -> 滑条选时间与分值 -> 输入备注 -> 保存

### Flow B：晚间复盘
复盘页 -> 点击生成复盘 -> 查看波峰/波谷与总结 -> 完成

### Flow C：长期洞察
洞察页 -> 切换周/月 -> 查看趋势 + 触发器排名 -> 调整行为

## 异常处理
- 网络中断：本地队列暂存，恢复后自动同步。
- 文本为空：允许保存。
- 重复提交：以后端ID主键避免重复写入。
- 风险语义：展示安全提示文本。

## 非功能需求
- 首屏加载 < 2.5s
- 交互FPS >= 45
- 核心API P95 < 400ms
- 全接口鉴权 + 用户级数据隔离
