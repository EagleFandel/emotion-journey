## English

### ğŸ‰ Overview

`v0.1.0` is the first public release of Emotion Journey, delivering a runnable Web PWA from product planning to implementation.

Core positioning: **lightweight logging + strong visualization + rule-based AI reflection**.

### âœ¨ Highlights

- Mood logging: time (0-24) + score (-5~+5) + one-line note
- Daily curve visualization
- Daily reflection with peak/valley detection, tags, and gentle summary
- Long-term insights: weekly/monthly trends and trigger analysis
- Lightweight admin panel: lexicon, risk terms, metrics
- Privacy features: data export and account/data deletion

### ğŸ§± Tech Stack

- Frontend: Next.js 16 + TypeScript + Tailwind + Recharts
- Backend: Next.js Route Handlers
- Data layer: PostgreSQL + Drizzle ORM (with in-memory fallback)
- Testing: Vitest + Playwright scaffold
- Tooling: pnpm workspace + ESLint + CI workflow

### âœ… Quality Checks

- `pnpm typecheck` passed
- `pnpm lint` passed
- `pnpm test` passed
- `pnpm --filter @emotion-journey/web build` may fail in restricted networks due to Google Fonts fetching

### âš ï¸ Known Issue

- Build can fail if `next/font/google` cannot fetch remote font assets.
- Next patch release will switch to local font assets for deterministic builds.
